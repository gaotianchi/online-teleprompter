几个月前我在一所小学实习，刚好赶上教师们录课的时候，当地的硬件设备比较充足，教师们在录课的时候可以借助移动大屏幕提词，她们最初使用翻阅 word 文档的方式提词，后来改用专业的提词工具，几个月来我们一直依靠在线提词器网站 https://cn.teleprompter-online.com/ 辅助录课任务，而这个项目正是深度使用该网站过后亲身实践的结果。
